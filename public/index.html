        Variables & functions

        Functions: sin cos tan asin acos atan atan2 exp log sqrt abs floor ceil round mod min max sign

        Constants: pi, e, i (imaginary unit)

        Power: e^x, x^2   Complex: Re(...) and Im(...) take the real / imaginary part, e.g. (1+Re(e^(i*(5x+t))))/2. A complex result without Re/Im uses its real part.

        Each channel should evaluate to a value in [0, 1] — values outside are clamped. t is time in seconds.

        Probing: tap a square (2D) or type coordinates above to inspect one grid point — the plot shows each channel's raw value from t−5 to t+5 seconds, with min/max reference lines.

        How to use

        Create a pattern

        On My Patterns, tap + New Pattern. Pick 2D or 3D, a grid size (40×40 is a good start), and a cell shape — squares, triangles, circles, or hexagons. Then hit Create.

        Write the equations

        Each of R, G, B is the brightness (0–1) of that color at every cell, as a formula using t (seconds) and the cell's position: x, y (z in 3D), or polar r, theta (phi in 3D) — mix them freely. Edit the text and the pattern updates live; a typeset version appears under each box.

        Try (1+sin(10r - 3t))/2. Functions like sin, cos, sqrt, e^x work, plus complex numbers: i, with Re(...) / Im(...) to get a real value back.

        Helper variables

        Tap + Add variable and type e.g. u = sqrt((x-1/2)^2 + y^2). Then use u inside R, G, B. Variables can use earlier variables.

        Time controls

        Pause / Play freezes the animation, t = 0 restarts it, and the t box is editable — type a value to jump to that exact moment.

        Inspect a point

        Tap the pattern (2D) to select a cell, or type coordinates in the x y z / r θ φ boxes (they snap to the nearest cell). A plot shows each channel's value at that cell from 5 s before to 5 s after now. Grid turns on cell borders, axes, and the coordinate indicators.
